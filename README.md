# VINNITY

## What is Vinnity?
Vinnity is a private chat app that can be used on servers or on your own computer. Our mission is to provide a secure and seamless communication platform that allows you to connect with your friends and communities privately. Whether you're setting up a server for your team or using Vinnity on your personal device, we prioritize your privacy and security.

## How to use?

You can use it on your own private server or you can use it in your personal computer. You should download the Vinnity and setup it to chat with others or create your own chat server.

### How to setup it for server?
#### To start setup file:
``$ python setup.py``

It'll control your operating system and if you have docker. If you don't have locker it'll download docker with you package manager. And it will add the file direction into path so you can use it easly.
