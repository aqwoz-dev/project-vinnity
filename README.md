# VINNITY

## What is Vinnity?
Vinnity is a private chat app that can be used on servers or on your own computer. Our mission is to provide a secure and seamless communication platform that allows you to connect with your friends and communities privately. Whether you're setting up a server for your team or using Vinnity on your personal device, we prioritize your privacy and security.
___
## How to use?

You can use it on your own private server or you can use it in your personal computer. You should download the Vinnity and setup it to chat with others or create your own chat server.

___
### How to setup it for server and personal computer?
### Linux:
+ ``$ git clone https://github.com/aqwoz-dev/project-vinnity.git``
#### Then after that run:
+ ``$ cd project-vinnity && sudo chmod +x setupforlinux.sh``
### Windows: 
Just execute the file with Administator:
>setupforwindows.bat
---

## Commands and how to run:

* It must've added into your PATH. So you can just run to open a server to chat:
* ``vinnity --server --on``
And then you can just run this to close the server:
* ``vinnity --server --off``
Or if you just wanna join someones server and chat(The server should have vinnity like you have.) just run:
* ``vinnity --joinserver``
